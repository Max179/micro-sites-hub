## Indie Hackers / HN Post

**Title:** Show HN: Free Meeting Cost Calculator — see the true dollar cost of any meeting

**Body:**

I kept seeing Reddit threads about meeting fatigue — engineers, managers, and PMs frustrated by back-to-back meetings with no visibility into what they actually cost. Calendar tools show time, not money. So I built a simple, free meeting cost calculator.

The idea: enter attendees, average salary, and duration. The tool calculates the real cost including a 1.4× loaded rate for benefits/overhead (per BLS data) and a 23-minute recovery time per person (based on Dr. Gloria Mark's UC Irvine research on context switching). Most calculators skip the recovery cost, which can add 30%+ to the true meeting expense.

Design principles:
- First screen is the calculator, not marketing copy
- No sign-up, no tracking, no clutter
- Works on mobile and desktop
- Results update instantly

A 6-person weekly sync at $85K average salary? $475/meeting, $24,700/year. A daily standup for 8 people? $26K/year. Seeing these numbers makes it a lot easier to justify cutting meetings that could be async messages.

Built with vanilla HTML/CSS/JS — no frameworks, no build step. Deployed on GitHub Pages.

Try it: https://max179.github.io/site-20260706-meeting-cost-calculator/
